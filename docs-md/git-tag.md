# Git Tag Management

Ellie provides subcommands to manage git tags.

## Create Tag

Creates a new tag.

**Usage:**

```
ellie git tag create
```

You will be prompted for the tag name.

## List Tags

Lists all tags in the repository.

**Usage:**

```
ellie git tag list
```

## Delete Tag

Deletes a tag.

**Usage:**

```
ellie git tag delete
```

You will be prompted for the tag name.
